End-to-End Speech Recognition

Recurrent Networks with Connectionist Temporal Classification Loss

● Leaderboard distance reduction

What it does

  • Built a sequence to sequence model that predicts phonemes directly from raw audio.
  • Used stacked Long Short Term Memory and Gated Recurrent Unit layers as the encoder.
  • Trained with Connectionist Temporal Classification Loss, which removes the need for frame level annotation of the audio.
  • Applied Beam Search Decoding at inference time to improve the predicted phoneme sequences.
  • Used Spectrogram Augmentation during training to make the model more robust.
  • Achieved a measurable reduction in Levenshtein Distance on the competitive class leaderboard.
